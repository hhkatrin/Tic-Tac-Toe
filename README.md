# Tic Tac Toe

A web-based Tic Tac Toe game built with HTML, CSS, and JavaScript.  
The project includes multiple game modes, a round system, and an AI opponent based on the Minimax algorithm.

---

## Live Demo

https://hhkatrin.github.io/tic-tac-toe/

---

## Features

- Player vs Player mode
- Player vs Computer mode
- AI opponent using Minimax algorithm (optimal play)
- Customizable number of rounds per match
- Score tracking across rounds
- Win detection with highlighted winning combination
- End screen with final results
- Restart functionality
- Responsive layout for desktop and mobile devices
- Basic sound effects for interactions

---

## How to Play

1. Open the game in a browser
2. Select the number of rounds
3. Choose a game mode (PvP or vs Computer)
4. Start the game
5. Players take turns placing X and O on the grid
6. The match continues until all rounds are completed
7. Final score is displayed at the end

---

## AI Logic

The computer opponent uses the Minimax algorithm to evaluate all possible moves and choose the optimal one.  
This makes the AI unbeatable when playing optimally.
